## NB功能介绍

* 快速创建Nutz应用,[初始化工具Maker](https://get.nutz.io)
* 嵌入式web服务(jetty/tomcat/undertow),可打包成单一jar文件
* 基于starter的自动配置体系,只需要添加maven依赖,即可自动发现并加载
* 能满足90%以上常见需求的默认配置,无需过多的自定义
* 以开放的心态与国内开源团体合作,优先集成国产项目
* 活跃的社区及稳健的发布周期,推进项目一直前进
* 提供swagger api文件自动生成
